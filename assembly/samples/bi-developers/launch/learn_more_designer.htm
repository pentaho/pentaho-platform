Report Designer

Report Designer is a graphical design tool created for advanced report designers who require full control over data access, layout, grouping, calculations, charting, and formatting for pixel-perfect reports. Report Designer and the Pentaho Reporting engine empower you to:

  • Quickly design sophisticated reports using drag-an-drop
  • Report against a broad set of data sources including relational, OLAP, and XML
  • Run reports in a variety of formats including PDF, HTML, Excel, and RTF
Report Designer includes a step-by-step wizard to guide new report designers through the report creation process.
 
 

To launch the Pentaho Report Designer and begin designing your own reports:

Windows

From the Windows Start menu, select Start > Programs > Pentaho > Pentaho Reporting > Report Designer

Linux

Run startdesigner_linux.sh found in the report-designer directory under your root Pentaho installation directory
